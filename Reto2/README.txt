# 🏥 Gestión de Veterinaria - "🐶🐱" 

Este es un sistema de gestión veterinaria basado en consola, desarrollado en Python. Permite registrar clientes y sus mascotas, agendar citas, actualizar o cancelar consultas, y llevar un historial clínico de los animales.

## 📌 Descripción del Proyecto

La veterinaria 🐶🐱 ha solicitado una solución digital para modernizar su administración. Actualmente, la información se gestiona manualmente, lo que genera pérdida de datos y dificultades en la organización. Este programa soluciona estos problemas implementando un sistema basado en **Programación Orientada a Objetos (POO)**.

### ✨ Funcionalidades

✅ Registro de clientes con información de contacto y dirección.  
✅ Registro de mascotas con nombre, especie, raza y edad.  
✅ Programación, actualización y eliminación de citas.  
✅ Historial clínico con seguimiento de servicios prestados.  
✅ Menú interactivo en consola para una fácil gestión.  
✅ Validación de datos y manejo de errores.  

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** Python 3  
- **Paradigma:** Programación Orientada a Objetos (POO)  
- **Interfaz:** Consola (modo texto)
